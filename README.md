# MWC Chat — Чат для майнеров

Стилизованный чат для майнеров MWC с регистрацией и каналами.

## Функции

- Регистрация и авторизация
- Каналы: general, mining, trading, hardware, offtopic
- Статусы онлайн/офлайн
- Тёмная тема в стиле криптовалют
- Honeycomb анимация на фоне
- LocalStorage для хранения данных

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/FuckFiat/mwc-chat.git

# Открыть index.html в браузере
```

## Использование

1. Откройте `index.html` в браузере
2. Нажмите "Регистрация"
3. Введите имя пользователя и пароль
4. Начните общаться!

## Каналы

| Канал | Описание |
|-------|----------|
| #general | Общий чат |
| #mining | Майнинг MWC |
| #trading | Трейдинг |
| #hardware | Железо и ASIC |
| #offtopic | Болталка |

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- LocalStorage API
