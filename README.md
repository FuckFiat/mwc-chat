# MWC Chat — Чат для майнеров

Стилизованный чат для майнеров MWC с регистрацией, темами и онлайн-панелью.

## Функции

- **Регистрация и авторизация** — LocalStorage для хранения данных
- **7 тем оформления** — тёмная, светлая, золотая MWC, IRC, терминал, Termius, Matrix
- **Онлайн пользователи** — отдельная панель справа
- **Только #general** — один чат для майнеров
- **Без ботов** — только реальные пользователи
- **Адаптивный дизайн** — работает на мобильных

## Темы

| Тема | Описание |
|------|----------|
| 🌑 Тёмная | Классическая тёмная тема |
| ☀️ Светлая | Светлая тема для дневного использования |
| 🪙 Золотая MWC | Золотые тона в стиле MWC |
| 💻 IRC | Классический IRC-стиль (зелёный текст) |
| ⬛ Терминал | Стиль терминала с `$` в начале |
| 🔐 Termius | Стиль SSH-клиента Termius |
| 🟢 Matrix | Matrix-стиль с неоновым свечением |

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/FuckFiat/mwc-chat.git

# Открыть index.html в браузере
```

## Использование

1. Откройте `index.html` в браузере
2. Нажмите "Регистрация"
3. Введите имя пользователя и пароль
4. Начните общаться!

## Технологии

- HTML5
- CSS3 (CSS Variables, Flexbox)
- JavaScript (ES6)
- LocalStorage API
